🎵 YouTube Integration Setup

To use your own YouTube playlists, you need to set up the YouTube Data API:

  1. Go to Google Cloud Console → APIs & Services → Credentials
  2. Click "Create Credentials" → "API Key"
  3. Select "YouTube Data API v3"
  4. Copy the generated API key (looks like AIzaSyC...)
  5. Open config.js and replace:
API_KEY: 'YOUR_API_KEY_HERE'

With your actual API key:

API_KEY: 'AIzaSyC...' // Your real API key

Then reload the page and click "Sign in with Google" to access your playlists!
